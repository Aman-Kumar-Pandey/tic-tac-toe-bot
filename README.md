# Tic Tac Toe – Easy/Hard AI

A lightweight, browser-based Tic Tac Toe game with adjustable AI difficulty.  
Perfect for beginners exploring **game logic, AI, and DOM manipulation**.  

The project is fully offline-ready, responsive, and playable directly in a modern browser.  
Designed for learning, experimentation, and fun.

⚠️ Note: This is a client-side game. Bugs, AI quirks, or logic gaps may exist—feel free to explore and contribute.

---

## Features

- Classic 3×3 Tic Tac Toe gameplay
- Player vs AI (X = user, O = AI)
  - Easy AI: random moves
  - Hard AI: Minimax algorithm
- Match progress tracking using progress bars
- Responsive design for desktop and mobile
- Reset round or reset entire match
- Highlight winning combination
- Sound effects on win
- Toggle difficulty (Easy / Hard)
- Fully offline, single-file playable

## Usage

No installation required.  

1. Open a version folder inside `Versions/v1.0/`  
2. For **standalone**, open `index.html`  
3. For **modular**, open `index.html` (CSS and JS are separate)  
4. Play directly in any modern browser
