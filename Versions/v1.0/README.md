# Tic Tac Toe – v1.0

Version 1.0 is the **first stable release** of the Tic Tac Toe game.  
It includes both **Easy and Hard AI modes**, match progress tracking, and responsive offline gameplay.

This version is delivered in **two formats**: a standalone build and a modular build.

---

## Available Builds

### 1. Standalone Build (`single-file/`)

**TicTacToeBot.html**

- Contains HTML, CSS, and JavaScript in a **single file**  
- Fully portable and offline-ready  
- Ideal for quick use, sharing, or learning game logic  

---

### 2. Modular Build (`modular/`)

**index.html + style.css + script.js**

- Same functionality as the standalone build  
- Code separated for better readability and maintainability  
- Perfect for learning, modification, or future extension  

---

## Core Functionality

- Player vs AI gameplay  
  - User plays X  
  - AI plays O  
  - Hard mode uses Minimax  
  - Easy mode chooses random moves  
- Detects wins, draws, and ongoing games  
- Highlights winning combination  
- Match progress bars (first to 100%)  
- Reset round or entire match  
- Toggle difficulty using a switch  
- Sound effect for wins  

---

## Browser Compatibility

Tested on modern Chrome, Edge, and Firefox.  
Requires JavaScript to be enabled.

---

## Design Notes

- No external libraries or frameworks  
- Offline, client-side only  
- Focused on simplicity, readability, and learning  
- Minimal UI for clarity and gameplay focus
